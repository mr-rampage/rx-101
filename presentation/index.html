Reactive eXtensions 101

A library that provides a way of performing asynchronous operations on event streams and data streams.

RxJava is comparable to Akka and Spring Reactor. RxJs is comparable to Bacon.js and Kefir.js

Who uses it:

  • Java 9
  • Javascript (in draft for ES7)
  • Netflix
  • Angular 2
  • Github
  • Couchbase

Support

Java 6+, IE6+

Use cases:

  • Asychronous & event based computation (UI)
  • Consume asynchronous data (Web sockets, web workers)
  • Reduce blocking on web services (Micro-services)
  • Concurrent programming
  • Parallelisable
  • Composable
